# Flask Full-Stack App (Frontend + Backend + MongoDB + Docker)

This project is a simple full-stack web application built using **Flask**, **MongoDB**, and **Docker**.  
It demonstrates how a Flask frontend communicates with a Flask backend API, which stores data in MongoDB.
